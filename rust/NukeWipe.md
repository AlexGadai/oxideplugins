**NukeWipe **allows you to wipe your server while running and with style.

**Chat Commands (permission: nukewipe. use):**


* 
**/nuke **starts a nuclear explosion (wipe)


**Permissions:**


* nukewipe.use


****Config file:****

````
{

  "Settings": {

    "Time Until Explosion": 20.0

  }
}
````
