**Easy Broadcast** allows you to broadcast a message to the chat.

**Commands:**


* 
**/bcast <message> **- Broadcast a message


**Permissions:**


* **broadcast.use**



**Config file:**

````
{

  "Display": "<color=blue>{title}</color>: {message}",

  "Title": "Server"
}

 
````
