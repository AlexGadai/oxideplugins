**Easy Reload** reloads your plugins with just an in-game command!

-> Simple, but may be useful!
**Permissions (OXIDE):**


* canReload


**Commands:**


* 
**/reload** - Reloads all installed plugins
* 
**/reload [PluginTitle]** - Reloads plugin [**PluginTitle**] (not filename!)
